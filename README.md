# code_buster_django
'www.codematters.codes'

Install  Codebuster in devices

```bash
  pip install -r requirements.txt
```




## Run Locally

Clone the project

```bash
  gh repo clone praTeek271/code_buster_django
```

Go to the project directory

```bash
  cd code_buster_django
```

Start the server

```bash
  python manage.py runserver
```
